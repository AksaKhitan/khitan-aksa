# Undangan Digital Khitan — Aksa Juliansyah

Website undangan digital premium (Tasyakuran Khitan) — HTML, CSS, dan JavaScript murni. Tanpa framework, tanpa build, langsung jalan di GitHub Pages.

## Cara Pakai

1. **Ganti foto** — timpa file `assets/img/aksa.jpg` dengan foto Aksa (rasio potret, mis. 1000×1250 px).
2. **Tambahkan musik** — letakkan file `assets/music/background.mp3` (instrumen/sholawat lembut bebas hak cipta).
3. Buka `index.html` di browser untuk pratinjau.

## Upload ke GitHub Pages

1. Buat repository baru di GitHub, mis. `undangan-aksa`.
2. Upload **semua isi folder ini** (jangan folder-nya, tapi isinya: `index.html`, `css/`, `js/`, `assets/`).
3. Buka **Settings → Pages → Branch: `main` / root → Save**.
4. Tunggu ±1 menit. Website tampil di `https://username.github.io/undangan-aksa/`.

## Undangan dengan Nama Tamu

Tambahkan `?to=Nama%20Tamu` di akhir link:

```
https://username.github.io/undangan-aksa/?to=Bagus%20Kustiawan
```

Maka di halaman pembuka muncul **"Kepada Yth. Bagus Kustiawan"**.
Spasi ditulis `%20` atau gunakan `+`.

## Struktur

```
/
├── index.html
├── css/      → style.css, animation.css
├── js/       → script.js, countdown.js
└── assets/   → img/aksa.jpg, music/background.mp3, icon/favicon.svg
```
